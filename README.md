# 🛰️ Nexus-Eng: The Engineering Second Brain

**Nexus-Eng** is an open-source productivity ecosystem specifically engineered for the rigorous demands of technical fields. It moves beyond simple note-taking by integrating project management (WBS), a technical logic repository (Logic Vault), and career development (Career Hub) into a single, cohesive interface.

## 🧠 The Vision

As engineers, we deal with a massive cognitive load. Technical information is often fragmented across code repositories, local files, and scattered documentation.

**Nexus-Eng** centralizes this data through three fundamental pillars:
1.**Logic:** Persistent storage for snippets, design patterns, and "Aha!" moments.
2.**Management:** Structured project breakdowns using **WBS (Work Breakdown Structure)** principles.
3.**Career:** Systematic documentation of technical challenges to ace high-level interviews.

---

## ✨ Core Features (MVP)

### 📂 WBS Engine (Project Structuring)

* **Recursive Hierarchy:** Break down complex software or hardware projects into infinite sub-tasks.
* **Progress Tracking:** Real-time visualization of module completion and project health.

### 📜 Logic Vault (Technical Memory)

* **Snippet Repository:** Save code blocks with syntax highlighting and intelligent tagging (Python, C++, SQL, etc.).
* **Knowledge Graph:** Link related technical concepts to build a personalized web of engineering knowledge.

### 💼 Career Hub (Professional Edge)

* **STAR Method Tracker:** Log technical challenges using the *Situation, Task, Action, Result* framework to prepare for top-tier interviews.
* **Portfolio Automator:** Organize academic projects into professional case studies.

## 🛠️ Tech Stack

* **Frontend:** Next.js (App Router) + Tailwind CSS + Shadcn/UI.
* **Backend:** Supabase (Auth, PostgreSQL, Edge Functions).
* **Type Safety:** TypeScript for a robust and maintainable architecture.
* **Icons & Assets:** Lucide React for consistent technical iconography.
